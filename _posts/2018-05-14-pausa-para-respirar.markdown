De folga nesta segunda-feira, o elenco da Ponte Preta pode, enfim, aproveitar um período sem compromissos por Copa do Brasil ou Série B no meio da semana. Como volta a jogar apenas no sábado, contra o Atlético-GO, em Bragança Paulista, a Macaca tem tempo para descansar os principais jogadores e se preparar para o próximo desafio.

 
 
 

A última vez em que ficou sem atuar no meio de semana foi entre 21 e 28 de abril, no espaço entre a vitória sobre o Criciúma e a derrota para o Londrina. Desde então, foram cinco jogos divididos entre sábado/domingo e quarta/quinta. **Veja a lista abaixo** :

 
 
 

- 28/04 (sábado) - Ponte Preta 0 x 1 Londrina
- 02/05 (quarta) - Ponte Preta 0 x 1 Flamengo
- 05/05 (sábado) - Guarani 2 x 3 Ponte Preta
- 10/05 (quinta) - Flamengo 0 x 0 Ponte Preta
- 13/05 (domingo) - Vila Nova 0 x 0 Ponte Preta

Agora, Doriva e sua comissão técnica ganham um período para colocar todo o elenco na mesma condição física. O único jogador que precisa de atenção especial do departamento médico, mas que deve ser liberado para atuar, é o lateral-esquerdo Orinho, com um hematoma no músculo posterior da coxa esquerda.
